Client Stock Statement

3PL accountability statement — per client, per product: opening → receipts → issues → adjustments → closing over a period. Closing is the live on-hand balance; opening is derived so the statement reconciles. Reads the DB.txns[] audit log.

Statement parameters
⟳
▾Product UoM Opening Receipts (+) Issues (−) Adjustments (±) Closing Notes

Receipts = receive / transfer-in / returns. Issues = dispatch / transfer-out. Adjustments = signed quantity adjustments & count corrections in the period.
